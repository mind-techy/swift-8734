This collection focuses on building **Zero Trust smart home systems** using:

- **Swift** for native Apple ecosystem development  
- **Matter Protocol** for interoperability across IoT devices  
- **Zero Trust Security** to ensure continuous authentication and secure device communication  

The goal is to create **secure, scalable, and interoperable smart home controllers** that do not rely on implicit trust, aligning with modern IoT security standards.
